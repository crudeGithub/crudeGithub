## 🦸‍♂️ **Chapter 1: Origin Story - The Developer**

Forget clean corporate designs! This portfolio is built as a **messy, sketchy, comic-book-styled paper prototype**. Every section is enclosed in its own comic panel, slightly tilted to give that authentic *"sketched on a napkin at 3 AM"* vibe.

### 📊 **Character Stats:**
> **Class:** Gameplay Programmer 🎮  
> **Level:** Senior Dev 🧙‍♂️  
> **Specialty:** You Imagine, I Make it ✨  
> **Weakness:** Tea Shortage 🫖  

### 🎭 **Key Features of the Lair:**
- 🌙 **Night Mode:** Toggle between an ink-on-paper look and a stealthy dark mode.
- 🖱️ **Dangerous Cursors:** A custom interactive cursor script complete with a **Death Counter**! *Watch your clicks!*
- 💥 **Comic Book UI:** Panels, speech bubbles, sketchy buttons, and dynamic sounds (`popsound.mp3`). POW!

---

## 🚀 **Chapter 2: The Missions - Dynamic Projects**

All "Missions" (projects) are loaded dynamically using **Supabase**. They are presented inside comic panels within a scrollable grid. 

- **Categorized Tabs:** Filter your main quests vs side quests effortlessly. 
- **Interactive Modals:** Play WebGL games *directly* in the browser using custom iframes (Portrait & Landscape support). 
- **Dynamic Media:** Hover over a mission to trigger its preview GIF or video! 

---

## ⚡ **Chapter 3: Power-Ups - Tech Stack**

What fuels this prototype?

* **HTML5 & CSS3:** Vanilla CSS tailored with variables for quick theming, CSS Grid & Flexbox, and custom sketchy scrollbars!
* **Vanilla JavaScript:** ES6+ logic handling routing, modals, audio, and state caching without heavy frameworks. 
* **Supabase Client:** Ready-to-go BaaS setup to pull categories, project lists, and skills from a database. 
* **Formspree:** Handling the "To-Be-Continued" contact form seamlessly.
* **Google Fonts:** Powered by *Patrick Hand* & *Architects Daughter* for that authentic handwriting style. ✍️

## 🧰 **Chapter 4: The Utility Belt - Setup**

Want to run this locally? It's as easy as equipping a level 1 sword!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/portfolio-paper-prototype.git
   ```
2. **Enter the dungeon:**
   ```bash
   cd portfolio-paper-prototype
   ```
3. **Launch the game:**
   Just open `index.html` in your favorite web browser! Or use **Live Server** in VS Code.
4. **Link your own Supabase:**
   Modify `supabase-client.js` with your own database credentials if you want to swap out the missions and skills!
